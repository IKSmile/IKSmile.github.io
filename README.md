## ✉️ Email Integration (EmailJS)

To activate the contact form:

1. Create an account → https://www.emailjs.com/

2. Get your credentials:

   * SERVICE_ID
   * TEMPLATE_ID
   * PUBLIC_KEY

3. Configure inside `main.js`:

```javascript
emailjs.init("YOUR_PUBLIC_KEY");

emailjs.sendForm(
  "YOUR_SERVICE_ID",
  "YOUR_TEMPLATE_ID",
  form
);
```
